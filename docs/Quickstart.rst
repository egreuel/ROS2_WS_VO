Requirements
------------

- ROS2 humble
- Ubuntu 22.04
- Unity 2021.3.15f1
- MARUS simulator
- requirement.txt

Change directory to asv_path_planner and install the requirements from the requirements.txt file:

.. code:: bash
    
    $ pip install -r requirements.txt
